Detailed description:
Module to avoid a mobile robot to lost wifi connectivity.This module registers the latest values of the wifi signal strength (see module Wifi_Monitor) and in the case that it falls under a certain value,it returns to the place that has the higher wifi strength in its list.The objective is to avoid the robot to lost connectivity permanently, recovering it by commanding the robot to move to a know location with wifi coverage.
