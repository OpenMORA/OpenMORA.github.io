Detailed description:
ICP-based SLAM application that taking as input a rawlog file with odommetry and laser scans, generates the resulting map (image, simplemap, etc).The path to the rawlog-file to be processed can be specifyed by parameter or via a MOOS variable.This module is a slightly modifiction of the MRPT based ICP SLAM application (icp-slam_main.cpp) from Jose Luis Blanco ClaracoFor more information see http://www.mrpt.org/Application:icp-slam
