Detailed description:
Speech synthesis (TTS), in different languages, using the free Festival program.This module requires installing the "festival" Text To Speech package to work at run-time.However, it can be compiled without problems even without festival installed in the system.The "festival" App is designed to work under linux systems, so in windows use Cygwin.For more info on Festival TTS see: http://www.cstr.ed.ac.uk/projects/festival
