About OpenMORA¶

Open Mobile Robot Arquitecture (OpenMORA) is a lightweight distributed architecture for mobile robots based on MOOS and MRPT. It provides off-the-shelf modules for common robotics platforms and sensors, dataset logging, MonteCarlo Localization, reactive navigation, etc.
